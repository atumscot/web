Welcome to NPT Scotland: v0.1 Fast Track

Last Updated: 2023-03-12

Network Planning Tool for strategic cycle network planning: Fast Track layer.

This is an early version of the tool and the the results will change through the course of 2023.

You may need to clear your browser cache to see the latest updates. The results of this this Fast Track version are based only on the 2011 Census travel to work data. Commuting patterns have changed since the pandemic, and not everyone commutes, the results should be interpreted accordingly. Future versions of the tool will include other trip purposes.

The results are publicly available. The main intended audience is local authorities undertaking cycle network planning to support more evidence-based and transparent investment, scheme location, and design decisions.

Provide feedback:

To report issues with the tool, open an issue in the public issue tracker (requires GitHub registering a GitHub account).
