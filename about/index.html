Intellectual Property and Open Source Policy

The background intellectual property underlying the NPT tool belongs exclusively to its owner, building on the Propensity to Cycle Tool (PCT) for England and Wales and with new methods developed by the University of Leeds, CycleStreet Ltd, and others. Like the PCT, the NPT tool is open source and licensed under the terms of the AGPLv3 to encourage community contributions and ensure public benefit arising from public investment in the tool, as outlined below.

Non-transferable and non-exclusive rights are granted to background intellectual property are granted for the project's sole purpose. The arising intellectual property will owned by the University of Leeds, the funder Transport Scotland, or jointly, depending on who generated or developed it. The terms of any license agreement will be negotiated in good faith and will be fair and reasonable, taking into account the scientific and financial contributions of the University of Leeds and other parties.

The NPT Scotland project is open source, and the code is available on GitHub. The code is licensed under the Affero General Public License (AGPL) version 3.0. See the full license in the project's open source repositories on GitHub in the bullet points below:

  • Data processing and modelling code license
  • Web app code license

This means that you are free to copy and re-use the code. If you distribute the code or a modified version of the code, you must make the source code available under the same license. This is to ensure that the code remains open source for public benefit.

We encourage feedback and contributions to the project:
